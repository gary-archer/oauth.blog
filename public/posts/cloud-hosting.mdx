# Cloud Deployment Goals

Previously I completed this blog's mobile theme by explaining <a href='ios-code-sample-coding-key-points.mdx'>Code Details</a> for the iOS code sample. Next I explain how this blog's code samples use cloud infrastructure. First though, I articulate some goals and requirements for a cloud platform.

### Fast Time to Market

No waiting for hardware to be ordered or on other people.

### Fast Infrastructure

Softare has environments. No waiting on IT departments or for hardware to be delivered.

### Productive Development

Empower developers 

### High Availability

Behaviors like autohealing based on health checks and autoscaling based on metrics. Use cloud native under the hood.

### Future Proof Hosting

Deploy anywhere. Ability to switch providers or operate in new markets. Cloud native, where you do more deployment work but get richer control.

### Best Supporting Components

Freedom to choose the components you want.

### Cost Effective

Can be multiple ways to achieve it. Cheap option means giving up control.
